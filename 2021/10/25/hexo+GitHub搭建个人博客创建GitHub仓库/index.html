个人博客搭建是基于hexo+github，我们建立在本地的博客要是想要让别人看到你就需要上传到远程服务器，而GitHub就相当于我们的服务器，我们是将本地的博客上传到GitHub，别人通过你的GitHub地址访问到你搭建的博客。
超链接在这里👉👉👉静Yu的博客
下面我们就先创建一个仓库。

创建仓库

1️首先，我们需要有自己的GitHub账号，这里就不详细介绍如何创建账号了，大家可以自己从网上找一些资料学习一下，其实很简单的。如果有什么问题可以单独私信我。
2️首先打开GitHub是这样的。点击右上角的+号，new repository。

3️这里有需要注意的地方，是我图中标红框的地方，仓库的名字，这里仓库的名字不能随便写，格式为你的GitHub注册名字+github.io，
例如我的就是JingYu1205.github.io

4️接下来选中Add a README file ,创建阅读文件，然后直接点最下面的创建仓库就行了

5️到此我们的仓库就创建完成了，如图。
